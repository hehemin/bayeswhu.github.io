7.1 The many faces of regression

The many faces of regression

  • The term regression can be confusing because there are so many specialized varieties.

  • In addition, R has powerful and comprehensive features for fitting regression models, and the abundance of options can be confusing as well.

The many faces of regression

The many faces of regression

  • In this lecture, well focus on regression methods that fall under the rubric of ordinary least squares (OLS) regression, including:

    • simple linear regression,
    • polynomial regression,
    • and multiple linear regression.

  • In OLS regression, a quantitative dependent variable is predicted from a weighted sum of predictor variables, where the weights are parameters estimated from the data.

  • Ill describe how to use R functions to fit OLS regression models, evaluate the fit, test assumptions, and select among competing models.

7.2 OLS regression

Fitting regression models with lm()

  • In R, the basic function for fitting a linear model is lm().

  • The format is myfit <- lm(formula, data)

  • The formula is typically written as Y ~ X1 +X2 +...+ Xk

Fitting regression models with lm()

Testing a linear hypothesis

  • The standard regression output as provided by summary() only indicates individual significance of each regressor and joint significance of all regressors in the form of t and F statistics.

  • Often it is necessary to test more general hypotheses.

  • This is possible using the function linearHypothesis() from the car package.

  • Recall that a linear hypothesis is of the general form \[ R\beta=r \]

Testing a linear hypothesis

  • Suppose we want to test the hypothesis that the coefficient of height is 3.5.

    library("car")
linearHypothesis(fit, "height=3.5")
    ## Linear hypothesis test

Polynomial regression

  • The plot suggests that you might be able to improve your prediction using a regression with a quadratic term.

    fit2 = lm(weight ~ height + I(height^2), data = women)
summary(fit2)

  • The I() function treats the contents within the parentheses as R regular expression.

  • You need this because the ^ operator has a special meaning in formular that you dont want to invoke here.

Polynomial regression

  • In general, an nth degree polynomial produces a curve with n-1 bends.

  • To fit a cubic polynomial, youd use

    fit3 = lm(weight ~ height + I(height^2) + I(height^3), data = women)

  • Although higher polynomial are possible, Ive rarely found that terms higher than cubic are necessary.
